## 👀 Motivation [🔝](#-frontend-for-app-)

Para el frontend de nuestra aplicación de recursos humanos, elegimos React debido a su capacidad para crear interfaces de usuario interactivas y altamente dinámicas. React nos permite construir componentes reutilizables, lo que mejora la eficiencia en el desarrollo y facilita el mantenimiento a largo plazo. Además, su ecosistema de bibliotecas y herramientas nos ayuda a implementar funcionalidades avanzadas, garantizando una experiencia de usuario fluida y atractiva, fundamental para la satisfacción de los empleados y la optimización de los procesos de gestión del talento.

## 🛠️ Run Locally [🔝](#-frontend-for-app-)

👉🏻 To use locally Ruta Dorada, you need to:

Clone our repository

```sh
git clone https://github.com/No-Country-simulation/s17-16-t-node-react.git
```

Move into frontend folder and install dependencies

```sh
cd frontend | pnpm i
```

Run the project locally of the frontend

```sh
pnpm dev
```

## 🎚️ Scripts [🔝](#-frontend-for-app-)

- `pnpm start`: Run the application preview production client with next.
- `pnpm dev`: Run the application development client with next.
- `pnpm build`: Builds the production bundle for the web application.
- `pnpm lint`: Linting errors in the codebase automatically.
